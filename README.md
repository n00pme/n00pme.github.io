## Localize
1. [install rvm](https://rvm.io/)
2. type as below
	``` bash
	rvm install 2.3
	gem install jekyll bundler
	git clone https://github.com/kyshel/kyshel.github.io.git
	cd kyshel.github.io
	bundle install
	bundle exec jekyll serve --host=0.0.0.0
	```
